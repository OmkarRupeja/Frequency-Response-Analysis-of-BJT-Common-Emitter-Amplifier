## Frequency Response Analysis of BJT Common Emitter Amplifier

This project presents the design and simulation of a **BJT Common Emitter (CE) amplifier** using TINA-TI, with a focus on analyzing its frequency response characteristics.

The circuit employs a **voltage divider biasing network** to establish a stable operating point (Q-point), ensuring the transistor operates in the active region. Input and output coupling capacitors are used to block DC components, while an emitter bypass capacitor is implemented to enhance AC gain.

The amplifier is analyzed across a wide frequency range to study its behavior in three key regions: low-frequency, midband, and high-frequency. The simulation demonstrates the impact of reactive components on gain variation, highlighting cutoff frequencies and overall bandwidth.

### Key Features:

* Stable biasing using voltage divider configuration
* AC signal coupling and DC isolation
* Gain enhancement using emitter bypass capacitor
* Frequency response analysis (Bode plot)
* Identification of cutoff frequencies and bandwidth

### Tools Used:

* TINA-TI (Analog Circuit Simulation)

### Applications:

* Audio signal amplification
* Analog signal conditioning
* Pre-amplifier stages in electronic systems

This project serves as a foundational study in analog electronics, emphasizing the relationship between circuit design parameters and frequency-dependent performance.
